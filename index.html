Setup and Installation

Install and configure Visual Studio Code

There are a few installation, setup and login steps we need to complete before we can begin:

  • Install Visual Studio Code
  • Install a "Live Preview" extension for Visual Studio Code
  • Log in using a Google or a Department of Education email account and register for GitHub (to enable AI assistance)
  • Select an AI model to assist us

Vibe Code Project Examples

Vibe Code Project Examples

The prompt to generate each example application came from a maximum of four sentences:

  • Word Cloud Generator (QuickChart.io API)
  • Pokemon Search (PokeAPI)
  • What's for Dinner? (Recipe Search) (TheMealDB)
  • Weather by IP (IPAPI.co and Open-Meteo)
  • Star Wars Character Search (SWAPI - Star Wars API)
