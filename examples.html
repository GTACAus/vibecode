These are additional web projects demonstrating the use of APIs or edge computing platforms like Cloudflare Workers to deliver data-driven applications.

Project Summaries

Project Name Main Idea
Fuel Price Finder Displays current or recent fuel prices using an servosavers API.
Flinders VIC Tide Tracker Provides tidal forecasts (high/low tide times and heights).
Color-Number Converter Converts images into colour by number activities.
Ink Timer A temperature monitoring and countdown timing tool, for use with IBT-4XS temperature monitor.
EPG (Electronic Program Guide) Provides a guide for Melbourne TV programs, using live streams. Click the icons to watch a stream.

Fuel Price Finder

A Cloudflare Worker that dynamically fetches and displays information, likely related to current fuel prices in VIC.

Tide Tracker

An application that uses a marine data and BOM API to provide accurate, location-based forecasts and high and low tide times. There's a live stream to Flinders surfcam.

Color-Number Converter

A utility application designed to help users make a colour by number.

Ink Timer

A bluetooth temperature monitor app, includes as many timers as you would ever need.

EPG (Electronic Program Guide)

This application fetches and displays live Melbourne television or programming, similar to what you would find on a smart TV or streaming service.
