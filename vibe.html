Word Cloud Generator

Generates a word cloud from any text you paste into the box, using the QuickChart.io API.

Full Prompt:
Generate an html and javascript web app that just uses a single html document. I want to be able to paste a pile of text and have https://quickchart.io/wordcloud make a word cloud from it.

What's for Dinner? (Recipe Search)

Enter a main ingredient to get possible recipes to cook for dinner using TheMealDB API.

Full Prompt:
Generate an html and javascript web app that just uses a single html document. I want it to take a user query and use themealdb.com/api to return some possible recipes to cook for dinner. Filter by main ingredient API example: www.themealdb.com/api/json/v1/1/filter.php?i=chicken_breast

Weather by Internet Location

Uses your IP address to determine your location via IPAPI.co and then retrieves local weather using Open-Meteo.

Full Prompt:
Generate an html and javascript web app that just uses a single html document. I want it to search my ip via IPAPI.co and give me local weather using open-meteo API.
